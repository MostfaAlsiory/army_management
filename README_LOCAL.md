# دليل تثبيت نظام إدارة الجيش محلياً

هذا الدليل يشرح كيفية تشغيل النظام على جهازك الشخصي (Windows/Mac/Linux).

## المتطلبات الأساسية
1. **Node.js**: إصدار 18 أو أحدث.
2. **PostgreSQL**: قاعدة بيانات لتخزين البيانات.
3. **npm** أو **pnpm**: لإدارة الحزم.

## خطوات التثبيت

### 1. إعداد قاعدة البيانات
- قم بإنشاء قاعدة بيانات جديدة في PostgreSQL باسم `army_db`.
- احصل على رابط الاتصال (Connection String)، وعادة ما يكون بالشكل التالي:
  `postgresql://username:password@localhost:5432/army_db`

### 2. إعداد المتغيرات البيئية
- قم بإنشاء ملف باسم `.env` في المجلد الرئيسي للمشروع.
- أضف السطر التالي مع استبدال الرابط برابط قاعدة بياناتك:
  ```env
  DATABASE_URL=postgresql://username:password@localhost:5432/army_db
  ```

### 3. تثبيت التبعيات
افتح الطرفية (Terminal) داخل مجلد المشروع ونفذ الأمر التالي:
```bash
npm install
```

### 4. إعداد جداول قاعدة البيانات (Migrations)
نفذ الأمر التالي لإنشاء الجداول المطلوبة:
```bash
npm run db:push
```

### 5. تشغيل النظام
لتشغيل النظام في وضع التطوير:
```bash
npm run dev
```
سيفتح النظام على الرابط: `http://localhost:5000`

## الميزات المضافة حديثاً
- **الفرز المتقدم**: في كافة الجداول (الأفراد، الحضور، التقارير).
- **تصدير Excel**: تصدير البيانات مع احترام الترتيب والفرز.
- **التحضير الجماعي**: زر "تحضير البقية كحاضر" في صفحة الحضور لتسريع العمل.
